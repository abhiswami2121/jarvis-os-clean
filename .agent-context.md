# Jarvis-OS-Clean Repo Map (READ FIRST before editing)

## Where to find things
- React components: src/components/{dashboard, runs, jarvis, chat, assistant-ui}/
- Pages/routes: src/app/{chat, dashboard, runs, documents, api}/
- API routes: src/app/api/{agent, dashboard, runs, mvp, jarvis-proxy, conversations}/route.ts
- Stores/state: src/stores/ (zustand+immer)
- Lib utilities: src/lib/ (chat-store, jarvis-stream, canvas-tools)
- Hooks: src/hooks/
- Styles: src/app/globals.css (Aurora glass tokens)

## Editing rules
- If task says modify hook X, go DIRECTLY to src/hooks/ or src/lib/
- If task says modify component X, go DIRECTLY to src/components/<domain>/
- DO NOT scan raw HTML/CSS template files unless explicitly asked
- DO NOT read same file twice unless you wrote to it in between
